# Archrium Weakauras for 3.3.5a 
`Project Started at 23-May-2019`

# How to import a weakaura?
1- write `/wa` ingame 
2- Click new > import
3- Copy raw text of the weakaura from github
4- Paste ingame import screen

# Naming Conventions 

`!` Command
`*` PvP Related
`+` PvE / Raid Related
`--` Eat/Drink/Reagents
`-` General
`<classKey>\`` Class Specific
