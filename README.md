Weakaura Repository

# How to import a Weakaura?
1- write `/wa` ingame  
2- Click `new` then `import`  
3- Enter desired weakaura on page click raw, copy text  
4- Paste ingame import screen  

# Naming Conventions 
`! ` Command  
`!! ` Specific Raid  
`+` General PvP   
`+` General PvE   
`-` General Combat  
`-` Resting    
`<classKey>`\` Class Specific
